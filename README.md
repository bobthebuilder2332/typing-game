# Simple Typing Game

A simple and minimalistic typing game built with GDScript. Test your typing accuracy with a large bank of 999 words.

## Features

* Large word bank of 999 of the most common words in the English language
* Live stats tracker for submitted words, correctly spelt, and incorrectly spelt words
* Color indicator for spelling accuracy (green is correct, red is incorrect)
* Early warning feedback system (word changes to orange before submission if spelt wrong)
* Built with Godot Engine 4.6.2 and GDScript

## How to play

1. Itch.io [https://bobthebuilder233200.itch.io/simple-typing-game]
2. Run locally in Godot 4 engine
    * Clone this repository
    * Open repository in Godot 4
    * Click run button in top right (or F5 / Cmd + B)
